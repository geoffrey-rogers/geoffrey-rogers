## What I'm building

**AmpliVex** | [amplivex.io](https://amplivex.io)
My personal brand and venture exploring outcome-based agentic AI. Built on Next.js, deployed on Vercel, with a live lead capture pipeline. The thesis: software is shifting from seats to outcomes, and the sellers who understand both sides of that shift will lead it.

**OmniPulse AI**
A daily voice news briefing app that generates and narrates personalized audio. Built on Next.js and Supabase, with brief generation on Claude and speech via Deepgram Aura. Includes a full automated quality gate that pairs deterministic validators with an LLM grader to keep output reliable at scale.

**Job Command**
A custom CRM and operations platform I built to run a systematic job search. Node.js backend, MCP server integrations, automatic model routing with fallback chains, and end-to-end pipeline automation. A working example of designing real systems around the Anthropic API.
